# 🧠 Digital System Design (Verilog)

> 📘 Level 2 • Semester 1  
> 💻 Course Lab Repository  

---

## 🚀 Overview

This repository contains my **Verilog HDL implementations**, simulations, and lab assignments for the **Digital System Design** course.  
It covers fundamental digital circuits and design concepts using hardware description language.

---

## 🛠️ Topics Covered

- 🔹 Basic Logic Gates  
- 🔹 Half Adder & Full Adder  
- 🔹 BCD to Excess-3 Converter
- 🔹 Excess-3 to BCD Converter  
- 🔹 Binary Adder-Subtractor  
- 🔹 BCD Adder
- 🔹 Mod-10 Asynchronous (Ripple) Counter / BCD Counter / Decade Counter  
- 🔹 Asynchronous Up-Down Counter / Ripple Counter using JK FF (3-bit)


---

## ⚙️ Tools & Software

- 🧩 ModelSim (Simulation)
- 🧠 Verilog HDL
- 💻 Intel Quartus (optional)

---

## 📌 How to Run

1. Open **ModelSim**
2. Compile `.v` files
3. Run testbench
4. Observe waveform output
